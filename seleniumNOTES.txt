
1.  driver.get() = (for open any link)
    example: driver.get('https://google.com')

2.  driver.title() = (for get title of the page)
    example: print(driver.title)

3.  driver.current_url = (for get the page current url)
    example: print(driver.current_url)

4.  driver.page_source = (for get page source)
    example: print(driver.page_source)

5.  driver.close() = (for close the parent window[but others window are still there])
    example: driver.close()

    example: driver.find_element_by_xpath("//a[@name='q']")
